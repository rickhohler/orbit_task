## Reporting a Vulnerability

We take the security of OrbitTask seriously.

If you discover a security vulnerability within OrbitTask, please send an e-mail to your-email@example.com (Replace with actual email or instruction). All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues.**

### Process

1.  **Report**: Email the vulnerability details to us.
2.  **Acknowledge**: We will acknowledge receipt of your report within 48 hours.
3.  **Investigate**: We will investigate the issue and verify usage/exploitation.
4.  **Fix**: We will work on a patch to resolve the vulnerability.
5.  **Release**: We will release a security update.
