## 0.9.0+1

*   **Production Release**: First stable release of the orbit_task ecosystem.
*   **Core**: Full support for Android (WorkManager) and iOS (BGTaskScheduler).
*   **Web**: Support for web execution (Timer based fallback).
*   **Testing**: Comprehensive test suite passing.

## 0.0.1+1

* Initial release of OrbitTask.
* Unified Task Scheduler API for Android, iOS, and Web.
* Supports one-time and recurring tasks.
* Isolate-based background execution on Mobile.
* Web support via Timer fallback.
* Integration with `workmanager` for native persistence.
